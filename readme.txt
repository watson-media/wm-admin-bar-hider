== Description ==
The WM Admin Bar Hider is developed to easily hide the admin bar on the frontend with multiple settings. You can hide the admin bar based on a selected set of pages, user roles or just disable the admin bar completely everywhere for every users.

We will be adding more options to allow administrators to hide the admin bar for custom post types & more in the future versions.
 
== Installation ==
 
1. Upload wm-admin-bar-hider to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit the plugin's settings page and choose where you would like to hide the admin bar.
 
== Frequently Asked Questions ==
 
= How can i hide the admin bar for selected pages =
You can do this by visiting the following URL:
/wp-admin/options-general.php?page=wm-admin-bar-hider
& then select the pages you would like to hide the admin bar & just save. Thats it.
 
= How can i hide the admin bar for selected user roles.
You can do this by visiting the following URL:
/wp-admin/options-general.php?page=wm-admin-bar-hider
& then select the user roles you, for whom, you would like to hide the admin bar & just save. Thats it.
